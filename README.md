# PyPropogator
Python-based library using Runge-Kutta methods to model propogation and amplification of pulses in fibres, intended for use in modelling propogation of pulses in Ytterbium Doped Fiber <br />
Based on work: https://doi.org/10.1109/JLT.2007.909373 and https://doi.org/10.1038/srep34742 <br />
See notebookvisualiser.ipynb for examples of code output <br />
